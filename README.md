##  Weather Consultant
A responsive web application that provides personalized weather advice based on temperature inputs. This project demonstrates 
my understanding of JavaScript logic, CSS3 styling, and DOM manipulation.

##  Live Demo
https://lokeny2.github.io/Weather-Consultant/

## Features
- **Dynamic Advice:** Provides unique feedback based on temperature ranges.
- **Unit Conversion:** Automatically converts Celsius inputs to Fahrenheit for international users.
- **Input Validation:** Uses  "Guard Clauses"  to handle empty fields or invalid data.


## Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom properties, radial gradients, and hover transitions.
- **JavaScript (ES6+):** Arrow functions, template literals, and DOM manipulation.

## What I Learned
During this project, I mastered several programming concepts:
1. **Helper Functions:** Breaking logic into small, reusable pieces like `convertToFahrenheit`.
2. **Boolean Logic:** Managing multiple conditions using logical operators.
3. **The DOM:** Connecting JavaScript logic to HTML elements to create an interactive experience.
4. **Separation of Concerns:** Organizing code into three distinct files for better maintainability.
